# Agent Skills

Universal skills and prompts repository for AI agents, compatible with multiple platforms such as Claude, Cursor, Codex, and other LLM-based assistants.

## About

This repository contains a collection of skills, prompts, and configurations that can be used to enhance the behavior and capabilities of various AI agents. The goal is to create a set of reusable, platform-independent resources.

## Compatibility

The skills in this repository are designed to work with:

- **Claude** (Anthropic)
- **Cursor** (AI-powered IDE)
- **Codex** (OpenAI)
- **GitHub Copilot**
- Other LLM-based assistants
